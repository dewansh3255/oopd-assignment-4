# OOPD Assignment 4: University ERP

[cite_start]This project implements a generic University ERP system as part of the OOPD Monsoon 2025 course[cite: 1]. [cite_start]It demonstrates the use of C++ Templates, Multithreading, and advanced Data Structures[cite: 2, 7, 19, 24].

## How to Build and Run

This project uses a `Makefile`.

1.  **Compile:**
    ```sh
    make
    ```

2.  **Run:**
    ```sh
    ./university_erp
    ```

3.  **Clean:**
    ```sh
    make clean
    ```
